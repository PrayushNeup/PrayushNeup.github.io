GAME/WEB DESIGN PROJECTS

Virtual 3D Environment

In my second year, I created a first-person perspective, 3D immersive virtual environment in Unity. I used Autodesk Maya to create 3D low poly models which were imported into Unity to be used as objects in the virtual world and used Audacity for the audio.

Website Recreation

In my second year, I had to make a website with my group members using drag-and-drop website-building software. We decided to choose Wix to make a website that focuses on helping low-income Ontarians with discovering nutritious food sources. Later, I decided to recreate parts of that website, specifically the “Home” page,“Healthy Eating on a Budget” page and “Get involved” page using only HTML and CSS.
